# EcoSpark 🌱 — India's Gamified Environmental Learning Platform

EcoSpark is a modern, high-impact web application designed to gamify environmental education for students across India. By turning learning into an interactive journey, EcoSpark empowers the next generation to take real-world action against climate change.

![EcoSpark Preview](https://ecospark-1a4af.web.app/og-image.png)

## 🚀 Key Features

- **🧠 Interactive Quizzes**: Test your knowledge on climate change, waste management, and biodiversity to earn EcoPoints.
- **🌍 Carbon Footprint Calculator**: A scientifically-grounded tool tailored for the Indian context to measure daily environmental impact.
- **🏆 Real-world Challenges**: Complete tangible actions (like planting a tree or waste segregation) and verify them using Gemini AI.
- **🤖 Eco-Buddy (AI Chatbot)**: A friendly sprout-bot powered by Google Gemini to answer all your sustainability questions in real-time.
- **📜 Digital Badges & Leaderboard**: Earn unique badges (First Leaf, Quiz Master, etc.) and climb the national rankings.
- **⚡ Real-time Sync**: Full Firebase integration for seamless user progress across devices.

## 🛠️ Tech Stack

- **Frontend**: React 19 + Vite (for lightning-fast development)
- **Styling**: Modern Glassmorphism + Vanilla CSS3
- **Backend/Database**: Firebase Authentication & Firestore
- **AI Engine**: Google Gemini 1.5/2.5 Flash
- **Animations**: Framer Motion for smooth, premium transitions

## ⚙️ Setup & Installation

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/adityagotlost/Ecospark.git
    cd Ecospark
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Configure Environment**:
    Create a `.env.local` file in the root directory and add your Gemini API Key:
    ```env
    VITE_GEMINI_API_KEY=your_gemini_api_key_here
    ```

4.  **Run Locally**:
    ```bash
    npm run dev
    ```

5.  **Build for Production**:
    ```bash
    npm run build
    ```
